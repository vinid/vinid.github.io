Talk: Twitter-Demographer

Federico Bianchi
Code Video Follow
Image credit: Twitter Data

Abstract

Using Twitter data has now become an essential element of effective research. Political and social scientists are now making use of these data to extract meaningful insights from real world events. To help researchers, we have designed a tool that can be used to add additional information, such as user (binary) gender, disambiguated locations and even HuggingFace classifiers information to support their work.
